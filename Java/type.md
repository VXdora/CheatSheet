# Javaの型

## boolean
- true
- false

## char
16bit Unicode文字．
\u0000~\uFFFF

## byte
8bit整数 -128~-127.

## short
16bit整数．
-32,768～32,767．

## int
32bit整数.
-2,147,483,648～2,147,483,647.

## long
64bit整数.
-9,223,372,036,854,775,808～9,223,372,036,854,775,807.

longで入れたいなら，数値の後に`L`をつける．
```Java
long num = 39433204432234523L;
```

## float
32bit単精度浮動小数点数.

## double
64ビット倍精度浮動小数点数
