# bootstrap
CSSフレームワーク．
下手にCSSを自作するよりも，こっちのほうが便利かも？


## ボタン

bootstrap を使用しているならば，積極的に使いましょう．

class="btn btn-xxx"で指定できます．

なお，xxx の部分は，

- primary で青色
- secondary で灰色
- danger で赤色
- warning で黄色
- success で緑色

が指定できます．

### -outline-
`class="btn btn-outline-xxx`で枠だけ色がついたボタンを表示させることができる．

### lg, sm
`.btn-lg`や`.btn-sm`でボタンの大きさの変更が可能．
